# Go Chicken 🐔

Go Chicken is an enterprise-grade B&W Wholesaler Poultry Supply Chain Engine featuring:
- **FastAPI Backend:** WhatsApp Cloud API Webhooks, Dynamic Pricing API, Fleet Logistics, Khata Ledger & Ollama Local AI Intent Classification.
- **Next.js Wholesaler Dashboard:** Monochromatic B&W UI with real-time polling, optimistic UI rollbacks, and interactive management.
- **PostgreSQL + pgvector:** Vector similarity search and historical demand forecasting.

## Architecture & Deployment
- **Frontend:** Next.js application located in `/frontend` (Deployable to Vercel / Netlify).
- **Backend:** FastAPI application located in `/backend` (Deployable to Vercel / Railway / Render / AWS).
